# Лабораторная работа №3

---

### В данной лабораторной работе реализован не большой cli интерфейс

Запуск cli
```
python -m src.main
```

Интерфейс содержит заранее рандомно сгенерированный набор списков при запуске для тестов сортировок
```
lists
```


Чтобы увидеть все доступные команды, можно ввести команду 
```
help
```

Пример использования команд
```
Команда выводит отсортированный список из команды lists

quick_sort
```

Команды поддерживают флаг -b (он выводит время исполнения данной функции)

```
quick_sort -b
```

Так же команды поддерживают передачу списков для сортировки написанных вручную

```
Нельзя ставить пробелы внутри списка!

quck_sort -b 1,2,3,4,54,23,10,1
```

Cli поддерживает работу со stack и queue, команды выводят подсказки по работе с ними
- ```stack```
- ```queue```

Выход из Cli (любая из приведённых ниже команд)
- ```quit```
- ```break```
- ```exit```
- ```stop```
